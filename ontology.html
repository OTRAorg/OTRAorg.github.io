Metadata

The Ontology for the Transmission and Re-Use of Argumentative Patterns (OTRA) allows to explicitly represent connections between texts and arguments. OTRA is licensed under a Creative Commons Attribution-ShareAlike 4.0 International License. Created by Jacob Langeloh, based on HDN by Carlo Meghini, Valentina Bartalesi, Daniele Metilli, Filippo Benedetti and on the SAWS ontology by Anna Jordanous (knowledge modelling and CTS), Faith Lawrence, Keith Lawrence (developers) and Charlotte Tupman. Date: 2024-06-24

Citation Signalling

Subclass of E55 Type
Description Denotes the type of textual signalling, i.e. whether or not the text tells us it is referring to something external of itself.

Claim

Subclass of E73 Information Object
Description A speech act that includes assertion by the speaker that they consider this utterance to be true.

Comment

Subclass of Connective Claim
Description Comment added by an editor which connects the subject to a certain interpretation without pointing to an external resource.

Connective Claim

Subclass of Claim
Description A Connective Claim proposes that "knowledge about x can be enriched through considering y", where x is the subject of the claim (generally a text passage) and y is the object of the claim (a text passage, a motif, a location).
Equivalent Class Reference

Parallel Type

Subclass of E55 Type
Description Further specifies a Textual or Topical Parallel by assigning it a type. The type usually states how close the subject and the object are to each other from 'Verbatim', i.e. word-for-word identical, to a 'LooseRendering', which makes them very different.

Parallel in Other Language

Subclass of Textual Parallel
Description Subcategory of a Textual Parallel which signifies that the object (the text where the connection is drawn towards) is in another language than the subject (the text the connection is drawn from).

Parallel in Same Language

Subclass of Textual Parallel
Description Subcategory of a Textual Parallel which signifies that the object (the text where the connection is drawn towards) is in the same language as the subject (the text the connection is drawn from).

Rendering of Text

Subclass of Parallel in Same Language
Description Subclass of a Textual parallel in the same language, which indicates that something is expressed "in a related but different way".

Statement

Subclass of F1 Work
Description General, abstracted type of discursive statements, e.g., "Muhammad was not a true prophet" or "Jesus was not truly crucified", which can be realized in different concrete formulations.

Statement Expression

Subclass of F2 Expression
Description A concrete expression of an argument Archetype. Realized in statement formulations.

Textual Parallel

Subclass of Loci Paralleli
Description Subclass of a Connective Claim that indicates some textual similarity between the subject and the object, which can be specified by further subclasses and type assignments.

Topical Parallel

Subclass of Loci Paralleli
Description Type of Connective Claim that signals a Topical Parallel between the subject and the object that thereby claims no textual overlap. Most Textual Parallels will also be Topical Parallels, but not vice versa.

Variant of Text

Subclass of Version of Text
Description Subclass of Textual Version that indicates how closely related two passages of text are. Aligned with the SAWS predicate "isVariantOf".

Verbatim of Text

Subclass of Parallel Type
Description Type of Textual Parallel that indicates a very close connection between subject and object. Aligned with the SAWS predicates "isVerbatimOfText".

Version of Text

Subclass of Textual Parallel
Description Subclass of a Textual Parallel that is aligned with the SAWS predicate "isVersionOf".

Explicit Signalling

Subclass of Citation Signalling
Description Subclass of Citation Signalling, which indicates that the author of a text makes it explicit that some citation or re-use is happening. Can be further specified by types.
